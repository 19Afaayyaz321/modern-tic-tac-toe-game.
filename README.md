Overview
This is a fully interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. It includes Player vs Player mode and Player vs Robot mode with adjustable difficulty levels. The design is colorful and responsive, and the game features sound effects and background music for an engaging experience.

üöÄ Features
- Two Game Modes: Player vs Player and Player vs Robot (AI)
- AI Difficulty Levels: Easy (random), Normal (strategic), Hard (unbeatable minimax algorithm)
- Scoreboard to track wins across rounds
- Custom player names
- Sound Effects: Applause and cheer when someone wins
- Background Music embedded via Vocaroo
- Pause/Resume game control with visual feedback (red = pause, green = resume)
- Reset and Stop buttons for game control
- Responsive design for desktop and mobile
- Stylish UI with gradient background and smooth hover animations
- Winner highlighting with celebration animations

How to Play
1. Enter your name (optional) for Player X
2. Choose game mode:
   - 2 Player: Play against a friend (enter Player O name)
   - Robot Mode: Play against AI (select difficulty: Easy, Normal, or Hard)
3. Click on empty cells to place your mark (X or O)
4. First player to get 3 marks in a row (horizontal, vertical, or diagonal) wins!
5. Use Pause to pause/resume, Reset to clear the board, Stop to end the game
6. Scores are tracked across multiple rounds

AI Difficulty Explained
- Easy: AI picks random moves. Perfect for beginners!
- Normal: AI uses smart strategy (wins if possible, blocks your win, takes center/corners)
- Hard: AI uses minimax algorithm. Plays optimally and is nearly impossible to beat!
